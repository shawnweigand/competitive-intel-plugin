# Claude Marketplace Plugins by Shawn Builds AI

Welcome to the collection of Claude Marketplace plugins built by [Shawn Builds AI](https://www.youtube.com/@shawnbuildsai). These tools are designed to help you leverage Claude's capabilities for research, automation, and productivity.

## 🚀 Available Plugins

### [Competitive Intel](file:///Users/shawnweigand/GitHub/shawnbuildsai-claude-marketplace/plugins/competitive-research/README.md)
Weekly competitive intelligence for your business. Research competitor websites, surface recent posts, product updates, pricing changes, and upcoming events.

## 🛠️ Installation

Each plugin in this repository can be installed into Claude:

1. Navigate to the specific plugin folder.
2. Follow the installation instructions in the plugin's `README.md`.
3. Usually involves downloading the `.plugin` file and installing it via the Claude interface.
